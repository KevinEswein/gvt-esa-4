Dokumentation

Das linke Objekt ist ein Möbiusband, das durch eine Parametrisierung in 3D erzeugt wird. (Colorierung abhängig von der z-Tiefe)
Ein Möbiusband ist eine einseitige Fläche mit nur einer Kante und einer Fläche. In diesem Kontext wird es durch eine parametrische Gleichung in 3D erzeugt, die eine Schleife mit einer halben Drehung darstellt. Es ist ein klassisches Beispiel für eine nicht-orientierbare Fläche.

Das Objekt rechts ist ein Hyperboloid, das ebenfalls durch eine Parametrisierung in 3D erzeugt wird. (Colorierung abhängig von der z-Tiefe)
Ein Hyperboloid ist eine doppelt gekrümmte Fläche, die durch eine Hyperbel erzeugt wird, die um eine ihrer Achsen rotiert. In diesem Kontext wird es durch eine parametrische Gleichung in 3D erzeugt, die eine symmetrische, sattelförmige Struktur darstellt.
